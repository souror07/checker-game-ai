🚀 Features

Smart AI opponent, optimized decision-making, and classic checkers rules.

🧠 How the AI Works

The AI evaluates all possible moves using the minimax algorithm and reduces unnecessary calculations using alpha-beta pruning to choose the best move.

📘 About This Project

This project is a practical implementation of artificial intelligence in board games. It shows how algorithms can simulate thinking and strategy.
